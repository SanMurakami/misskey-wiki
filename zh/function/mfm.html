Misskey Flavored Markdown（Misskey 特有的 Markdown 语法）简称 MFM ，是可以在Misskey实例上许多地方能够使用的特殊格式标记语言。你可以在这里查询可以使用的 MFM 语言关键词。

你可以在这些地方使用 MFM

  • 帖子

在帖子管理菜单中可以将包含 MFM 源码的文本拷贝到剪贴板，便于确认代码。

  • 个人资料简介
  • 聊天消息
  • 页面
  • 实例说明与实例通知等场合

经 MFM 处理过的文本能够在同由 Misskey 驱动的服务器上显示，但不是所有效果都会显示在其他支持 ActivityPub 的服务器上。

基本的 MFM

其实，这种 MFM 也用在 Misskey 中的提及和话题标签等熟悉的地方。

提及

使用 @ + 用户名可以提及任意用户

@ai

话题标签

使用井号# + 话题名称可以向帖子上添加话题标签。

#Misskey自我介绍部

当您在 @ 符号或 # 符号后输入一个字符时，会出现一个候选列表。 此列表会在您键入时缩小范围，因此您在列表中找到您要查找的用户或主题标签时，可以使用箭头键或 Tab 将其选中，然后按 Enter 键完成输入。这样便利的功能请务必试一试。
